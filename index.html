Help & FAQ

Getting Started

EZ Quiz Web is a simple offline quiz tool supporting multiple‑choice, true/false, yes/no, and matching questions. No sign‑up required!

How Do I Use AI to Create Questions?

Use the ✨ Prompt button in the header (or press Ctrl/Cmd+P). Enter a Topic and a Length, click Copy, paste into your AI tool (e.g., ChatGPT), then paste the output back into the textbox.

What Question Types Are Supported?

  • MC: Multiple‑choice (single or multi‑answer, e.g., A,C).
  • TF: True/False (T/F).
  • YN: Yes/No (Y/N).
  • MT: Matching (e.g., 1‑A).

How Do I Enter Questions?

Use the format TYPE|Question|Options|Answer (e.g., MC|What is 2+2?|A) 3;B) 4|A). See the built‑in set for examples.

What If a Question Is Skipped?

Invalid lines are skipped. Check the diagnostic message under the input for hints.

Can I Save My Quizzes?

Copy the textarea contents or save them to a .txt file; quizzes aren’t saved in‑app.

How Do I Start?

Enter questions and click “Start Quiz.” You can enable a timer or switch themes in Settings. Tip: press Enter to start and to advance; Backspace goes back.

Manual Prompt (Advanced)

Prefer a copy‑paste prompt? Use this template and replace {N} and {TOPIC}:

You are QuizMaker. Create exactly {N} quiz lines about {TOPIC}.

Allowed line types (choose any mix):

MC: MC|Question?|A) Opt1;B) Opt2;C) Opt3;D) Opt4|A

For multi-answer MC: separate letters with commas, no spaces (e.g., A,C).

TF: TF|Statement.|T OR TF|Statement.|F

YN: YN|Question?|Y OR YN|Question?|N

MT: MT|Prompt.|1) L1;2) L2;3) L3|A) R1;B) R2;C) R3|1-A,2-B,3-C

Hard constraints:

Output only the quiz lines. No headings, numbering, quotes, explanations, or extra text.
Exactly {N} non-blank lines. One question per line. No empty lines.
ASCII characters only (no smart quotes). No trailing spaces.
Follow the exact separators: pipes |, semicolons ;, and commas , as shown.
MC has exactly A-D options; answer is letters only (e.g., A or A,C). TF uses T/F. YN uses Y/N.
MT includes at least 2 left items and 2 right items, with a correct mapping (e.g., 1-A,2-B,...).

Begin output now (no extra text before or after the lines).
