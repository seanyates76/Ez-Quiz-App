Security Policy
===============

Supported versions
------------------
- The latest `main` branch is the supported development baseline for this repository.
- The production mirror may lag slightly behind while filtered sync and release review are in progress.

Reporting a vulnerability
-------------------------
- Preferred: GitHub Security Advisories for the production mirror: <https://github.com/seanyates76/Ez-Quiz-App/security/advisories/new>
- Alternative: email **ez.quizapp@gmail.com** with the subject `Ez-Quiz security`
- Do not open public issues for undisclosed vulnerabilities.

Scope
-----
This repository includes:
- client-side application code
- Netlify Functions
- deployment and CI configuration
- development tooling and mirror automation

Please note:
- some alerts in the development repo may come from local or CI tooling dependencies rather than deployed app runtime paths
- reports should focus on real exposure, not just advisory count
- do not include secrets, private tokens, or personal data in reports

Response expectations
---------------------
- acknowledgement within 3 business days
- status updates every 7 days while the issue is active

Good-faith testing
------------------
Please avoid:
- denial-of-service testing
- destructive data modification
- spam or excessive automated requests
- public disclosure before remediation is coordinated
