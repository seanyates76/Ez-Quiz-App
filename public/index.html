Help & FAQ

Getting Started

  1. Paste quiz lines into the Editor (left) or use ✨ Prompt Builder to generate them.
  2. Click Generate to parse and preview; then Start to begin.
  3. Use Enter for Next, Backspace for Previous during the quiz.

Accepted Line Formats

  • MC|Question?|A) Opt1;B) Opt2;C) Opt3;D) Opt4|A (multiple: |A,C)
  • TF|Statement.|T (or F)
  • YN|Question?|Y (or N)
  • MT|Prompt.|1) L1;2) L2|A) R1;B) R2|1-A,2-B

Delimiter rules are strict: use the shown pipes |, semicolons ;, and labels.

Using AI (✨ Prompt Builder)

Open ✨, set a topic and length, then Copy. Paste into ChatGPT/Gemini and bring the generated lines back into the Editor.

Saving & Settings

  • Theme and timer settings persist automatically across reloads.
  • Reset App (Settings → Advanced) clears saved settings and reloads.

Shortcuts

  • Open Prompt Builder: Ctrl/Cmd + P
  • During quiz: Enter = Next, Backspace = Previous

Troubleshooting

  • Parser errors? Check delimiters and labels (e.g., A), 1)).
  • Start disabled? Ensure at least one valid question parsed.
  • Timer? Enable in Settings; set duration as mm:ss for countdown.
  • Reset? Use Reset App to clear saved theme/timer.
