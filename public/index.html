Help & FAQ

Quick Start

  1. Paste or load a .txt file into the Editor, or click Use Demo Set.
  2. Click Generate to parse and preview; then click Start to begin.
  3. During the quiz, use Enter for Next and Backspace for Previous.

Question Formats

  • MC|Question?|A) Opt1;B) Opt2;C) Opt3;D) Opt4|A (multiple: |A,C)
  • TF|Statement.|T (or F)
  • YN|Question?|Y (or N)
  • MT|Prompt.|1) L1;2) L2|A) R1;B) R2|1-A,2-B

Use the exact delimiters shown: pipes |, semicolons ;, and labels like A) or 1).

Quiz Controls

  • MC: click options (multiple selections allowed if the answer has multiple letters).
  • TF/YN: use the on‑screen buttons or keys (T/F, Y/N).
  • Keyboard: Enter/→ for Next, Backspace/← for Previous.

Results & Retake

  • Missed questions are shown first by default; toggle Show all to see everything.
  • Retake repeats the quiz; use the caret to Retake missed only.
  • Your answer list shows both the letter and the answer text.

Timer & Settings

  • Enable a timer in Settings; choose Stopwatch or Countdown.
  • Set duration as mm:ss for countdown.
  • Optional: Require answer to advance and Auto‑start after generate.

Saving & Reset

  • Theme and timer preferences save automatically and persist across reloads.
  • Use Reset App in Settings to clear saved preferences.

Advanced

  • Prompt Builder ✨: open, enter a Topic and Length, then Copy to get a ready‑made prompt you can use in your AI tool; paste the resulting lines into the Editor.
  • Generate from Topic: the Generate button can fetch quiz lines if a backend is configured; otherwise, paste lines or use the Demo.
  • Formatting tips: no extra numbering or commentary; one question per line; MC answers can be single (A) or multiple (A,C).

Troubleshooting

  • Parser errors: check delimiters/labels and remove extra bullets or numbering.
  • Start disabled: ensure at least one valid line was parsed in the Editor.
  • Timer not showing: enable it and set a duration (for countdown).
  • Stuck/needs refresh: use Reset App in Settings.
