Help

Getting Started

  • Enter a Topic, set Length and Difficulty, and select Generate.
  • Click Start to begin. No account is required.

Creating Quizzes

  • Use Options to configure Timer, Theme, and Question Types.
  • Open Options → Advanced options to access the Prompt Editor and tools.
  • Supported types: Multiple Choice (single/multiple), True/False, Yes/No, Matching.

AI Generation

  • Generate sends Topic, Length, Difficulty, and selected types to the AI.
  • If nothing returns, try again shortly (rate limits), use Demo Set, or paste your own lines.

Taking Quizzes

  • Keyboard shortcuts: Enter/Right = Next; Backspace/Left = Previous.
  • Answer hotkeys: MC (A–D), True/False (T/F), Yes/No (Y/N).
  • Require answer to advance: Settings → Quiz Behavior.

Timer & Settings

  • Timer: Stopwatch or Countdown (mm:ss) in Settings.
  • Preferences are stored locally. Use Reset App to clear.

Results & Retakes

  • Results show score, time, and a Missed/All filter (Missed first by default).
  • Retake missed only: use the caret next to Retake.

Import/Export

  • Import: Options → Advanced options → Load .txt, or paste into the editor.
  • Export: Copy from the editor or mirror into a .txt file.

Data & Privacy

  • Settings are stored in your browser.
  • AI generation sends Topic and settings to the configured provider. See Privacy for details.
  • Works offline once loaded (PWA). AI generation requires connectivity.

Troubleshooting

  • Start is disabled: no valid questions parsed yet. Use Demo Set or check the status under the editor.
  • Timer isn’t visible: enable it in Settings and set a duration for countdown.
  • Support banner doesn’t appear: extensions may block third-party widgets. The Support button is always available.

Parser Reference

Formats
  • MC|Question?|A) Opt1;B) Opt2;C) Opt3;D) Opt4|A (multiple: A,C)
  • TF|Statement.|T (or F)
  • YN|Question?|Y (or N)
  • MT|Prompt.|1) L1;2) L2|A) R1;B) R2|1-A,2-B

Use the exact delimiters shown: pipes |, semicolons ;, and labels like A) or 1).
